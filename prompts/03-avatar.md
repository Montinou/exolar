# Cosmic Avatar

## Purpose
Default user profile picture.

## Technical Specifications
- **Dimensions**: 256 x 256 px
- **Background**: Solid #12121A (Horizon Dark)
- **Format**: PNG
- **Style**: Abstract, ethereal, character-based

## Image Generation Prompt

```
An abstract, cosmic representation of a user avatar on a solid dark background #12121A.
Center subject: A "being" made of condensed stardust and light. No facial features.
The silhouette assumes a human-like head-and-shoulders shape but is composed of a dense cluster of Event Violet #8B5CF6 and Quasar Cyan #00E5FF particles.
The "core" of the head is brighter, suggesting a mind at work.
Around the silhouette, very faint orbital rings in Nebula Gold #FFB700 suggest an electron cloud or a halo.
The lighting is soft and internal—the figure glows from within.
The style is reminiscent of a high-end sci-fi interface element, not a cartoon or a 3D render of a person.
Minimalist, recognizable at small sizes (32px).
```

## Style Notes
- **Anonymity**: Must not look like a specific gender or ethnicity.
- **Energy**: The user is represented as "intelligence," not "body."
- **Avoid**: Eyes, mouths, realistic hair, robotic metal plating.

## Consistency Reference
Uses the secondary palette (Violet/Cyan) to differentiate user content from system content (Gold/White).
